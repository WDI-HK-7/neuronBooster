# neuronBooster
Assessment tool to improve your coding skills

## Install Steps
```
$ npm install express --save
$ npm install express-generator -g
$ express -c less --hbs .
$ npm install
```

## Run the app
```
$ DEBUG=myapp ./bin/www
$ open -a "Google Chrome" http://localhost:3000
```
